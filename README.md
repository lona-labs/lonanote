# LonaNote

### run

```
cd ui
flutter run
```

### build desktop

```
cd ui
flutter build windows
flutter build macos
flutter build linux
```

### generate

```
cd ui

# generate code
flutter_rust_bridge_codegen generate

# generate icon
dart run flutter_launcher_icons
```

### install project

```
cd ui
flutter pub get

cd ui/cargokit/build_tools
flutter pub get
```
